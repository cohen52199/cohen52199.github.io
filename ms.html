Microsoft Office

Microsoft Office is an integrated suite of business software applications for computers. One Microsoft Office application I have a certification in includes Word, a widely used commercial word processor. Another powerful application is PowerPoint, used to create presentaions. Then, there is Excel, a spreadsheet program used to present tables of values that can be manipulated mathematically. Finally, a program used for both small and large database deployments is Access.
